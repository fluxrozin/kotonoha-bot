# 聞き耳型: 会話の状態判定用プロンプト

以下の会話履歴を見て、会話の現在の状態を判定してください。

【重要な前提】

- Discord の参加者は全て場面緘黙の当事者（自助グループのメンバー）です
- 会話の内容、文脈、雰囲気を総合的に判断してください

【判定すべき状態】

1. **ENDING（会話が終了しようとしている）**

   - 参加者が会話を締めくくろうとしている
   - 「これ以上」「以上です」「以上になります」「申し上げることはありません」など、会話を終了させる意図が明確
   - 会話の目的が達成され、自然に終了しようとしている
   - **ただし、単に話題が変わっただけの場合は ENDING ではない**

2. **MISUNDERSTANDING（参加者間の誤解が発生している）**

   - 参加者間で誤解や認識の相違が発生している
   - 「誤解」「勘違い」「違う」「そうではない」「間違い」などの表現が使われている
   - 参加者が誤解を解こうと説明している
   - 参加者間で意見の相違が表面化している
   - **決めつけ的な発言が発生している（「あなたは〜だと思っている」「〜という前提で話している」など、相手の意図や考えを勝手に決めつけている）**
   - **ただし、単なる意見交換や議論の場合は MISUNDERSTANDING ではない**

3. **CONFLICT（対立や緊張感が高まっている）**

   - 参加者間で対立や意見の相違が表面化している
   - 感情的になっている、不穏な空気が漂っている
   - 場が荒れてきている、緊張感が高まっている
   - ファシリテートや調整が必要な状況
   - **決めつけ的な発言が発生している（「あなたは〜だと思っている」「〜という前提で話している」など、相手の意図や考えを勝手に決めつけている）**
   - **ただし、建設的な議論の場合は CONFLICT ではない**

4. **ACTIVE（会話が順調に進行している）**
   - 会話が活発に進行していて、特に問題がない
   - 参加者同士で自然に会話が成立している
   - 上記のいずれの状態にも該当しない

【判定の優先順位（厳守）】

1. **最優先**: 会話が終了しようとしているか（ENDING）

   - 参加者が明確に会話を締めくくろうとしている場合は、必ず ENDING と判定してください
   - ただし、単に話題が変わっただけの場合は ENDING ではない

2. **次に重要**: 参加者間の誤解が発生しているか（MISUNDERSTANDING）

   - 参加者間で誤解や認識の相違が発生している場合は、MISUNDERSTANDING と判定してください
   - 決めつけ的な発言が発生している場合は、MISUNDERSTANDING と判定してください
   - ただし、単なる意見交換や議論の場合は MISUNDERSTANDING ではない

3. **次に重要**: 対立や緊張感が高まっているか（CONFLICT）

   - 参加者間で対立や意見の相違が表面化している場合は、CONFLICT と判定してください
   - 決めつけ的な発言が発生している場合は、CONFLICT と判定してください
   - ただし、建設的な議論の場合は CONFLICT ではない

4. **デフォルト**: 会話が順調に進行している（ACTIVE）
   - 上記のいずれの状態にも該当しない場合は、ACTIVE と判定してください

【重要な注意事項】

- **会話の内容と文脈を重視してください**
- **単純にキーワードが含まれているだけでなく、会話の全体の流れと雰囲気を判断してください**
- **参加者が会話を締めくくろうとしている場合は、必ず ENDING と判定してください**
- **参加者間の誤解が発生している場合は、MISUNDERSTANDING と判定してください**
- **決めつけ的な発言が発生している場合は、MISUNDERSTANDING または CONFLICT と判定してください（誤解を生む場合は MISUNDERSTANDING、対立や緊張感を高める場合は CONFLICT）**
- **対立や緊張感が高まっている場合は、CONFLICT と判定してください**
- **建設的な議論と対立を区別してください**

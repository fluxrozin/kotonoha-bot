# 聞き耳型: 同じ会話判定用プロンプト

以下の 2 つの会話を比較して、**同じ会話の続きか、別の会話か**を判定してください。

【重要な前提】

- Discord の参加者は全て場面緘黙の当事者（自助グループのメンバー）です
- 会話の内容、トピック、参加者、時間的な連続性を総合的に判断してください

【判定基準 - SAME と答える場合】
以下のような場合、同じ会話の続きとして判定してください：

1. **トピックが同じ、または関連している**

   - 同じ話題が続いている
   - 関連する話題に発展している
   - 会話の流れが自然に続いている

2. **参加者が同じ、または重複している**

   - 同じ参加者が会話を続けている
   - 新しい参加者が加わったが、トピックは同じ

3. **時間的な連続性がある（ただし、時間が経過していても返信の場合は同じ会話）**

   - 会話が自然に続いている
   - 短い間隔で会話が続いている
   - **時間が経過していても、古いメッセージへの返信の場合は同じ会話として扱う**

4. **会話の文脈が続いている**
   - 前の会話の内容を参照している
   - 前の会話から自然に発展している

【判定基準 - DIFFERENT と答える場合】
以下のような場合、別の会話として判定してください：

1. **トピックが全く異なる**

   - 全く違う話題に変わっている
   - 会話の流れが途切れている

2. **時間的な間隔が長く、かつ内容が全く関連していない**

   - 会話が長い間途切れていて、内容も全く関連していない
   - **ただし、古いメッセージへの返信の場合は、時間が経過していても同じ会話として扱う**
   - 時間情報は参考程度にし、内容を重視してください

3. **会話の文脈が切れている**

   - 前の会話との関連性が感じられない
   - 新しい会話が始まっている

4. **参加者が大きく変わっている**
   - 参加者が全く異なる（ただし、トピックが同じ場合は同じ会話として扱う）

【重要な注意事項】

- **会話の内容と文脈を重視してください**
- **時間的な情報は参考程度にし、内容を優先してください**
- **Discord では時間が経過していても古いメッセージに返信して会話を続けることがあります**
- **トピックが同じ、または関連している場合は、時間が経過していても同じ会話として扱ってください**
- **トピックが全く異なる場合は、時間が短くても別の会話として扱ってください**

【判定の優先順位】

1. **最優先**: トピックと内容の関連性
2. **次に重要**: 会話の文脈の連続性
3. **参考程度**: 時間的な間隔、参加者の変化
